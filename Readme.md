# 💼 Employee Attrition Prediction & Analytics System

## 📌 Problem Statement

Employee attrition is a major challenge for organizations. High attrition leads to increased hiring costs and loss of productivity. This project aims to predict whether an employee is likely to leave the company based on key factors.

---

## 🎯 Objective

* Analyze employee data to identify attrition patterns
* Build a machine learning model to predict employee attrition
* Develop an interactive dashboard for real-time predictions

---

## 🛠️ Tech Stack

* **Programming:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Visualization:** Matplotlib, Seaborn
* **Deployment:** Streamlit
* **Version Control:** Git & GitHub

---

## 📊 Features

* 📈 Exploratory Data Analysis (EDA)
* 🤖 Machine Learning Model (Random Forest)
* 📊 Feature Importance Visualization
* 🎯 Real-time Prediction using Streamlit
* 📉 Attrition Insights Dashboard

## ▶️ How to Run

### 1. Clone Repository

```
git clone https://github.com/MohammadAta-BIET7522/HR-Attrition-Project.git
cd HR-Attrition-Project
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Train Model

```
python train_model.py
```

### 4. Run App

```
streamlit run app.py
```

## 📊 Key Insights

* Employees with **low income** and **high overtime** are more likely to leave
* **Early-stage employees** have higher attrition
* Work-life balance plays a significant role in retention
